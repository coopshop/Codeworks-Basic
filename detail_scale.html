Scaling in the Cloud with Amazon Web Services

10:00 - 10:50

Eli White

Making your web application scalable is always a tricky task. This talk will center around how you can do this more easily on the cloud, specifically using Amazon Web Services such as EC2, RDS, S3 and more features. Amazon has a very rich feature set that makes having a scalable application easier than ever before.
